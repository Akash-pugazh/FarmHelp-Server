### Farmhelp Api

```
POST /api/v1/users
{
    userKey: "user key to fetch"
}
```

> Expects a body with userKey and returns the particular user details in object

---
<br>

```
GET /api/v1/users
```

> Returns all the users details from database
---

<br>


```
GET /api/v1/orders
```

> Returns all the orders details from database
